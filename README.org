* Overview

A simple calculator implementation built with Bison and Flex. This project serves as a hands-on learning exercise for understanding parser generators and lexical analyzers.

Based on the tutorial at https://cis.k.hosei.ac.jp/~nakata/lectureCompiler/YaccLex/index.html

* Getting Started

#+begin_src sh
git clone https://github.com/conao3/ccalc.y.git
cd ccalc.y
make
#+end_src

* Usage

Run the compiled calculator:

#+begin_src sh
./ccalc
#+end_src
